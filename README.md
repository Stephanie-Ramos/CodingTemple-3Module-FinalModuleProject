# JavaScript API Dashboard

A simple **API Dashboard** built with HTML, CSS, and JavaScript. This project fetches data from multiple public APIs and displays it in an interactive grid layout.

## Features

- 🐶 **Random Dog Image** – Fetch a random dog from the Dog API  
- 🐱 **Random Cat Image** – Fetch a random cat from The Cat API  
- ☀️ **Weather Info** – Displays current and hourly temperature for Los Angeles using Open-Meteo  
- and more
